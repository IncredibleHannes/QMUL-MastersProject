newtype K r x #

Definition of the quantifier type. The type variable R can be seen as a generalised truth value. For example for r = Bool, a function can be build with the type:

f :: [x] -> K Bool x

is returning true if an element exists for wich the function:

f2 :: (x -> r)

is returning true.
